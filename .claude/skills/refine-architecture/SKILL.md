# Refine Architecture Skill

## What to do?

- Review the most recent code changes in the repository
- Update or create `OVERVIEW.md` while reviewing
- Update the `ARCHITECTURE.md` file to reflect all changes

## When to do?

- Whenever I ask you to commit code changes
- Whenever I ask you to refine architecture explicitly
